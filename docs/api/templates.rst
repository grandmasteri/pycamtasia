Templates
=========

Save a project as a ``.camtemplate``, install template packages, and
instantiate new projects from them.  The :mod:`~camtasia.templates.behavior_presets`
module ships pre-built animation presets (reveal, fly-in, pop-up, etc.) while
:mod:`~camtasia.templates.lower_third` provides a ready-made lower-third title
template pre-flattened for Camtasia compatibility.

   :doc:`/guides/template-projects`
      End-to-end guide for cloning project structure and swapping media.

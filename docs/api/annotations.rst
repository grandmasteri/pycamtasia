Annotations
===========

Callout overlays and geometric shapes for highlighting regions of the canvas.
The :mod:`~camtasia.annotations.callouts` module builds callout definitions
(including sketch-motion callouts), :mod:`~camtasia.annotations.shapes`
provides shape primitives, and :mod:`~camtasia.annotations.types` defines the
annotation type constants.  Favourite annotations can be saved and loaded for
reuse across projects.

Captions
========

Caption attributes, dynamic caption styles, and accessibility validation.
:class:`~camtasia.timeline.captions.CaptionAttributes` controls font, colour,
position, and background for on-screen text.  Captions can be extracted to SRT
for translation and reimported, making localisation workflows scriptable.  The
accessibility validator checks contrast ratios and minimum font sizes.

Types
=====

Central enum and :class:`~typing.TypedDict` definitions shared across the
library.  The module exports 17 enums — including :class:`~camtasia.types.ClipType`,
:class:`~camtasia.types.EffectName`, :class:`~camtasia.types.TransitionType`,
and :class:`~camtasia.types.InterpolationType` — plus structured dicts such as
:class:`~camtasia.types.DropShadowParams` and :class:`~camtasia.types.ClipSummary`.
Import from :mod:`camtasia.types` rather than reaching into submodules.

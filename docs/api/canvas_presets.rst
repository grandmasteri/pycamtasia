Canvas Presets
==============

Preset canvas dimensions for common recording and publishing targets.  Use
these constants to set a project's width and height to standard resolutions
(1080p, 4K, vertical mobile, etc.) and to query platform-specific safe zones
so that titles and captions stay within visible bounds on any device.
